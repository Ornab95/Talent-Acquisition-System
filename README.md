# Talent Acquisition System

## Overview

The Talent Acquisition System is a web application designed to streamline the recruitment process at Southeast University. It provides a platform for HR to post jobs, manage applications, and for candidates to apply for job opportunities. The system is built using HTML, Tailwind CSS, JavaScript for the frontend, Spring Boot for the backend, and MongoDB as the database.

## Features

- **User Authentication:** Secure login and signup functionality.
- **Job Posting:** HR can post and manage job listings.
- **Application Management:** Track and manage candidate applications.
- **Responsive Design:** Fully responsive design to ensure usability on different devices.

## Technologies Used

- **Frontend:**
  - HTML
  - Tailwind CSS
  - JavaScript
- **Backend:**
  - Spring Boot
- **Database:**
  - MongoDB
